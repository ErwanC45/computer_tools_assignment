DATABASE GROUP PROJECT 

This is an ongoing group project for the Database Course.
In this project, we have to imagine and create a database for a client, starting from an initial design to the final design.
In this particular file, we created tables, inserted data, used various commands to create views, queries to get data from the table and so.

How to use it ?

To use this file you can run it with SQLPlus or SQL Developer, but it's best designed for SQLPlus.
